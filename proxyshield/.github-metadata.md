Repository:
ProxyShield

Description:
Self-hosted Layer-7 reverse proxy written in Go (standard library only) that learns each client's normal traffic pattern and auto-throttles anomalies. Includes rate limiting, a signature WAF, honeypot auto-bans, a circuit breaker, response caching, a real-time SSE dashboard, Prometheus metrics, and a full-stack demo app it protects.
